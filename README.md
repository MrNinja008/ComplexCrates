A useful PocketMine-MP plugin that allows you to create crates in-game!

# Commands
Main command: `/crate`
Sub commands:
- `create` - Creates a crate
- `edit` - Edits a crate
- `remove` - Removes a crate
- `list` - Lists all crates
- `givekey` - Gives key to player
- `givekey-all` - Gives key to all players
